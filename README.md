# filament-storage
Documentation and software for a 3D printing filament storage base on the IKEA Billy shelf

The idea for the solution is based on a [article by Heise Make](https://www.heise.de/ratgeber/Bastel-Projekt-Aufbewahrungsschrank-fuer-Filamentrollen-bauen-10243342.html) and the [YouTube video](https://youtube.com/watch?v=RGxapS3P5mA&si=o0nBTALMwI8owIox) they made about it.

So far, this is a work in porgress and not at all finished.

As a start and since I have next to no experience with microcontrollers and such things, I will start with the basic functionality. The current temperature and humidity is shown on the display and as long the air humidity is above a certain level, the dehumidifier is turned on to reduce it.
